# 2026-07-21 工作日志

## 1. 所有后端接口加 /api 前缀
- boss 要求所有接口统一加 `/api` 前缀。
- 方案：在 `platform-admin/src/main/resources/application.yml` 主配置加 `server.servlet.context-path: /api`（dev/prod 均继承）。**未改动任何 Java 代码**。
- 兼容性确认（无需改）：SaTokenConfig 的 `addPathPatterns("/**")` 与 exclude 列表匹配去 context-path 后的 servletPath；CorsConfig 的 CORS Filter 注册 `/**` 为 servlet 级、最高优先级，与前缀无关；StartupInfoRunner 已读 `server.servlet.context-path` 并打印访问地址。
- 踩坑修正：初版误把 `context-path` 加在 `spring.servlet`（原 `spring.servlet.multipart` 文件上传配置块）下，已纠正到 `server.servlet`（Spring Boot 标准 key），`multipart` 保留在 `spring.servlet` 下。python yaml 校验通过。
- 副作用提醒（前端侧，不在本仓库）：Swagger UI → `/api/swagger-ui.html`、api-docs → `/api/v3/api-docs`、文件直链 `/file/xxx` 实际为 `/api/file/xxx`；前端 axios `baseURL` 需设 `/api`。
- 另澄清：实际 `server.port` 是 **8085**（非 8080）；Dockerfile 历史记录里的 8080 为误记。
